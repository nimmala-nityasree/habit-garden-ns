# 🌱 Habit Garden

Habit Garden NS is a simple and interactive habit tracking web application designed to help users build consistency and maintain positive daily habits in a visually engaging way.

---

## 📖 About The Project

This project was built as part of my learning journey in web development and deployment using Lovable, GitHub, and VS Code.

The main idea of Habit Garden is to turn habit tracking into a visual “growth” experience, where consistency helps users stay motivated and disciplined.

---

## ✨ Features

- Add and manage daily habits
- Mark habits as completed
- Track daily consistency visually
- Clean and minimal UI
- Responsive design for different devices
- Simple and beginner-friendly workflow

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Lovable (for project generation)
- Git & GitHub
- VS Code
